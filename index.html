Palestra

Angular.js

GROUPBUDDIES

Angular.js é uma framework open-source escrita em JavaScript e desenvolvida pela Google que permite a criação de aplicações web dinâmicas direccionadas para o cliente (client-side). O foque desta passa por adicionar poder expressivo às páginas HTML que todos conhecemos, fazendo jus ao seu lema "Enhanced HTML Web Apps".
Assim, será Ronaldo Sousa, da Group Buddies, o responsável por dar a conhecer esta nova framework a todos os web developers desejosos por aumentar o seu conhecimento.

INMOTION

A InMotion vem dar a conhecer estes dois micro-dispositivos que estão a revolucionar o mundo da tecnologia quer pelo seu tamanho, quer pelo seu custo, quer pelo leque de funcionalidades oferecidas.
